# webpack-react

> 使用 Webpack 搭建 React 开发环境

[with Typescript](https://github.com/haledc/webpack-react/tree/ts)

## 实现功能

- ES6
- React
- Scss/css (modules / autoprefixer)
- Linter (ESLint + Prettier)

## 启动

```bash
# install node_modules
yarn
# or npm install

# development env
yarn dev

# production env
yarn build
yarn start
```
